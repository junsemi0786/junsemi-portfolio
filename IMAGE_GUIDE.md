# 이미지 관리 및 업로드 가이드

현재 홈페이지는 이미지 파일을 **직접 업로드하는 방식이 아니라**, 프로젝트 폴더에 이미지를 넣고 주소를 입력하는 방식으로 동작합니다.

## 1. 이미지 파일 추가하기
1.  컴퓨터에서 프로젝트 폴더의 `public/images` 폴더를 엽니다.
    - 경로: `.../engineering-platform/public/images`
2.  사용하실 이미지 파일(jpg, png 등)을 이 폴더에 복사해 넣습니다.
    - *팁: 파일명은 영문 소문자와 숫자, 하이픈(-)만 사용하는 것이 좋습니다 (예: `project-new.jpg`).*

## 2. GitHub에 변경사항 올리기 (배포)
이미지 파일을 추가했으므로, 이를 GitHub에 올려야 실제 사이트(Vercel)에도 반영됩니다.

1.  터미널에서 프로젝트 폴더로 이동합니다.
2.  아래 명령어를 입력하여 이미지를 업로드합니다:
    ```bash
    git add .
    git commit -m "Add new images"
    git push origin main
    ```
3.  Vercel이 자동으로 변경사항을 감지하고 약 1~2분 뒤에 배포를 완료합니다.

## 3. 관리자 페이지에서 이미지 등록하기
1.  홈페이지 관리자 페이지(`/admin`)로 접속합니다.
2.  실적(Case)이나 기술 역량(Expertise) 등록/수정 화면으로 이동합니다.
3.  **썸네일 이미지 경로** 입력란에 방금 올린 파일의 경로를 입력합니다.
    - 입력 형식: `/images/파일명.jpg`
    - 예시: `/images/project-new.jpg`
4.  저장하면 즉시 적용됩니다.

## 요약
- **이미지 추가**: `public/images` 폴더에 파일 넣기
- **배포**: `git push`로 업로드
- **적용**: 관리자 페이지에서 `/images/파일명` 입력
